# 익스턴십 - 라우팅 명세서

> **프로젝트**: OZ 오즈코딩스쿨
> **GNB 메뉴**: 커뮤니티, 질의응답 | 로그인, 회원가입 (비회원) / 프로필 아이콘 (회원)

---

## 1. 공통 레이아웃 (Layout)

| 요소                            | 설명                                                                                    |
| ------------------------------- | --------------------------------------------------------------------------------------- |
| **상단 배너**                   | 프로모션 배너 (예: "선착순 모집! 국비지원 받고 4주 완성")                               |
| **GNB (Global Navigation Bar)** | 로고(`OZ. 오즈코딩스쿨`), 커뮤니티, 질의응답 / 우측: 로그인·회원가입 또는 프로필 아이콘 |
| **챗봇 위젯**                   | 하단 우측 플로팅 — 새 채팅 시작, 이전 내역 불러오기                                     |

---

## 2. USER 라우팅

### 2.1 인증 (Auth)

| Route             | 화면명               | 설명                                                                                                         | 인증 필요 |
| ----------------- | -------------------- | ------------------------------------------------------------------------------------------------------------ | --------- |
| `/login`          | 로그인               | 카카오/네이버 소셜 로그인 + 이메일·비밀번호 일반 로그인. 아이디 찾기·비밀번호 찾기·일반회원 로그인 링크 포함 | ✗         |
| `/signup`         | 회원가입 (방법 선택) | 소셜 가입 또는 일반 가입 선택                                                                                | ✗         |
| `/signup/general` | 일반 회원가입        | 이메일, 비밀번호, 이름, 휴대폰 인증, 성별, 생년월일 등 입력 폼                                               | ✗         |

### 2.2 랜딩 페이지

| Route | 화면명      | 설명                                              | 인증 필요 |
| ----- | ----------- | ------------------------------------------------- | --------- |
| `/`   | 랜딩 페이지 | 서비스 소개. 질의응답·쪽지시험·커뮤니티 섹션 포함 | ✗         |

> 디자인 파일 내 `랜딩페이지_질의응답`, `랜딩페이지_쪽지시험`, `랜딩페이지_커뮤니티` 프레임은 동일 랜딩 페이지의 각 섹션(또는 탭) 뷰로 추정됩니다.

### 2.3 마이페이지 (My Page)

| Route                     | 화면명               | 설명                                                                                          | 인증 필요 |
| ------------------------- | -------------------- | --------------------------------------------------------------------------------------------- | --------- |
| `/mypage`                 | 마이페이지 - 내 정보 | 프로필, 개인 정보, 수강 중인 과정 표시. 좌측 사이드 메뉴: 쪽지 시험 · 내 정보 · 비밀번호 변경 | ✓         |
| `/mypage/edit`            | 마이페이지 - 수정    | 닉네임, 휴대폰 번호 등 개인정보 수정 (휴대폰 번호 인증 포함)                                  | ✓         |
| `/mypage/change-password` | 비밀번호 변경        | 현재 비밀번호 확인 + 새 비밀번호 입력                                                         | ✓         |
| `/mypage/withdraw`        | 회원 탈퇴            | 탈퇴 안내 모달 → 탈퇴 확인 메시지                                                             | ✓         |

### 2.4 쪽지시험 (Quiz)

| Route                  | 화면명                     | 설명                                                      | 인증 필요 |
| ---------------------- | -------------------------- | --------------------------------------------------------- | --------- |
| `/mypage/quiz`         | 마이페이지 - 쪽지시험 목록 | 응시 가능한 쪽지시험 목록 표시                            | ✓         |
| `/quiz/:quizId`        | 쪽지시험 - 시험보기        | 참가 코드 입력 후 시험 시작 (코드 불일치 시 에러)         | ✓         |
| `/quiz/:quizId/take`   | 쪽지시험 - 응시하기        | 문제 풀이 화면. 경고(시간 초과 등), 관리자 종료 알림 포함 | ✓         |
| `/quiz/:quizId/submit` | 쪽지시험 - 제출 완료       | 시험 제출 완료 안내                                       | ✓         |
| `/quiz/:quizId/result` | 쪽지시험 - 결과 확인       | 시험 결과 및 정답 확인                                    | ✓         |

### 2.5 질의응답 (Q&A)

| Route                   | 화면명               | 설명                                                                                                | 인증 필요                       |
| ----------------------- | -------------------- | --------------------------------------------------------------------------------------------------- | ------------------------------- |
| `/qna`                  | 질의응답 - 질문 목록 | 전체보기·답변완료·답변 대기중 탭. 검색, 카테고리 필터, 정렬(최신순/필터 등) 기능 포함               | ✗                               |
| `/qna?search={keyword}` | 질문 검색 결과       | 질문 목록에서 키워드 검색 적용                                                                      | ✗                               |
| `/qna/:questionId`      | 질의응답 상세 페이지 | 비회원: 질문 내용 + 댓글 불가. 회원: 답변 등록·수정·채택 가능. AI 답변 요청 및 결과 표시. 챗봇 연동 | 조건부 (읽기: ✗ / 답변·댓글: ✓) |
| `/qna/write`            | 질문 등록            | 대분류·중분류 카테고리 선택 + 제목·내용 입력. 예시 내용 제공                                        | ✓                               |
| `/qna/:questionId/edit` | 질문 수정            | 기존 질문 내용 수정                                                                                 | ✓                               |

**질의응답 상세 페이지 상태별 뷰:**

| 상태               | 뷰 설명                               |
| ------------------ | ------------------------------------- |
| 비회원 / 댓글 없음 | 질문 내용만 표시, 답변·댓글 작성 불가 |
| 회원 (질문자)      | 답변 채택 가능, 질문 수정 가능        |
| 회원 (답변자)      | 답변 등록·수정 가능                   |
| 채택 완료          | 채택된 답변 표시                      |
| AI 답변            | AI 답변 요청 → 로딩 → 결과 표시       |
| AI 챗봇            | 플로팅 챗봇 위젯을 통한 추가 질의     |

### 2.6 커뮤니티 (Community)

| Route                     | 화면명                 | 설명                                                            | 인증 필요                  |
| ------------------------- | ---------------------- | --------------------------------------------------------------- | -------------------------- |
| `/community`              | 커뮤니티 - 목록 페이지 | 게시글 목록, 검색, 필터, 정렬 기능                              | ✗                          |
| `/community/:postId`      | 커뮤니티 상세 페이지   | 비회원: 읽기 전용. 회원: 댓글 작성 가능. 작성자: 수정·삭제 가능 | 조건부 (읽기: ✗ / 댓글: ✓) |
| `/community/write`        | 게시글 작성            | 제목·내용 입력 (리치 텍스트 에디터)                             | ✓                          |
| `/community/:postId/edit` | 게시글 수정            | 기존 게시글 내용 수정                                           | ✓                          |

**커뮤니티 상세 페이지 상태별 뷰:**

| 상태   | 뷰 설명                                   |
| ------ | ----------------------------------------- |
| 비회원 | 게시글·댓글 읽기만 가능                   |
| 회원   | 댓글 작성 가능                            |
| 작성자 | 게시글 수정·삭제 가능 (삭제 시 확인 팝업) |

| ------------------------------ | ---------------------------- | --------------------------------------------------------- |

| 아이디 찾기 결과 모달          | `/login`                     | 조회된 이메일(아이디) 표시 / 에러 시 "일치하는 계정 없음" |
| 비밀번호 찾기 모달             | `/login`                     | 이메일 입력 → 인증 메일 발송 요청                         |
| 비밀번호 찾기 이메일 인증 모달 | `/login`                     | 이메일 인증 코드 입력                                     |
| 비밀번호 재설정 모달           | `/login`                     | 새 비밀번호 입력 + 확인. 완료 시 안내                     |
| 계정 상태 확인 모달            | `/login`                     | 비활성/잠긴 계정 안내                                     |
| 계정 복구 이메일 인증 모달     | `/login`                     | 비활성 계정 복구를 위한 이메일 인증                       |
| 수강생 등록 모달               | `/signup/register-student`   | 과정 선택 드롭다운 + 등록 확인                            |
| 카테고리 필터 모달             | `/qna`                       | 대분류·중분류 카테고리 선택 적용                          |
| 컨텐츠 정렬 모달               | `/qna`                       | 질문 정렬 순서 변경                                       |
| 질문 등록 입력 팝업            | `/qna/write`                 | 내용 미입력 시 안내 팝업                                  |
| 게시글 삭제 확인 팝업          | `/community/:postId`         | 작성자 삭제 시 확인 팝업                                  |
| 비밀번호 재설정 완료 팝업      | `/login/find-password/reset` | 비밀번호 변경 완료 안내                                   |
| 회원탈퇴 안내 모달             | `/mypage/withdraw`           | 탈퇴 전 안내사항 표시                                     |
| 쪽지시험 경고                  | `/quiz/:quizId/take`         | 시간 초과, 관리자 종료 등 경고                            |
| 참가코드 불일치                | `/quiz/:quizId`              | 잘못된 참가코드 입력 시 에러                              |

## 5. 전체 라우트 트리 (Route Tree)

```
├── /                                ← 랜딩 페이지
├── /login                           ← 로그인 (아이디 찾기, 비밀번호 찾기, 계정 복구는 모달)
│
├── /signup                          ← 회원가입
│   ├── /general                     ← 일반 회원가입
│
├── /mypage                          ← 마이페이지 (내 정보)
│   ├── /edit                        ← 정보 수정
│   ├── /change-password             ← 비밀번호 변경
│   └── /quiz                        ← 쪽지시험 목록
│
├── /quiz/:quizId                    ← 쪽지시험 시작 (참가코드 입력)
│   ├── /exam                        ← 시험 응시
│   └── /result                      ← 결과 확인
│
├── /qna                             ← 질의응답 목록
│   ├── /write                       ← 질문 등록
│   └── /:questionId                 ← 질의응답 상세
│       └── /edit                    ← 질문 수정
│
├── /community                       ← 커뮤니티 목록
│   ├── /write                       ← 게시글 작성
│   └── /:postId                     ← 커뮤니티 상세
│       └── /edit                    ← 게시글 수정

```

## 6. 접근 권한 요약

| 권한               | 접근 가능 경로                                                                                                               |
| ------------------ | ---------------------------------------------------------------------------------------------------------------------------- |
| **비회원 (Guest)** | `/`, `/login`, `/signup/**`, `/qna` (읽기), `/qna/:id` (읽기), `/community` (읽기), `/community/:id` (읽기)                  |
| **회원 (User)**    | 위 전체 + `/mypage/**`, `/quiz/**`, `/qna/write`, `/qna/:id/edit`, `/community/write`, `/community/:id/edit`, 댓글·답변 작성 |
| **관리자 (Admin)** | `/admin/**`                                                                                                                  |

## 7. 참고 사항

- **챗봇 위젯**: 전 페이지에 플로팅 형태로 존재하며, 별도 라우트가 아닌 컴포넌트로 관리합니다. 질의응답 상세 페이지에서 AI 답변과 연동됩니다.
- **반응형**: 디자인 기준 해상도는 1920×1080 (Desktop) 입니다.
- **소셜 로그인**: 카카오, 네이버 OAuth 연동이 필요합니다.
- **휴대폰 인증**: 회원가입, 마이페이지 수정 시 SMS 인증 플로우가 포함됩니다.
